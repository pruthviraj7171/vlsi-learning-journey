# Stage 1 — Digital Design Mastery 

This stage focuses on building strong fundamentals in digital electronics, which are essential for VLSI design and hardware engineering roles.

## Topics Covered

### Boolean Algebra
- Binary Numbers
- Logic Gates
- Truth Tables
- Boolean Expressions
- Boolean Laws
- De Morgan’s Laws
- Expression Simplification
- Minterm & Maxterm
- Canonical Forms (SOP, POS)
- Karnaugh Map (K-Map)

### Combinational Circuits
- Half Adder
- Full Adder
- Half Subtractor
- Full Subtractor
- Multiplexer (MUX)
- Demultiplexer (DEMUX)
- Encoder
- Decoder
- Comparator
- Code Converters

### Flip-Flops
- SR Flip-Flop
- D Flip-Flop
- JK Flip-Flop
- T Flip-Flop
- Characteristic Table
- Characteristic Equation
- Excitation Table
- Edge Triggering
- Clocked Operation

### Registers
- Register Concept
- Shift Registers
- SISO, SIPO, PISO, PIPO
- Universal Shift Register
- Data Storage in Registers

### Counters
- Ripple Counter (Asynchronous)
- Synchronous Counter
- Up Counter
- Down Counter
- Up-Down Counter
- Mod-N Counter
- Ring Counter
- Johnson Counter

### Finite State Machines (FSM)
- FSM Concept
- State Diagram
- State Table
- Moore Machine
- Mealy Machine
- Sequence Detector

### Timing Concepts
- Setup Time
- Hold Time
- Clock-to-Q Delay
- Propagation Delay
- Timing Diagrams
