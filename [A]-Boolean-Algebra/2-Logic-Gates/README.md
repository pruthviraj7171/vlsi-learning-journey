## Logic Gates 

Logic gates perform logical operations on one of more binary inputs to produce a single output

## Types of Logic Gates

- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate

## AND Gate
The and gate gives the output '1' when the both input are '1'.

## OR Gate 
The OR Gate gives the output '1' if atleast one input is '1'

## NAND Gate

The NAND gate is the **inverse of the AND gate**.  
Output is **0 only when both inputs are 1**.
