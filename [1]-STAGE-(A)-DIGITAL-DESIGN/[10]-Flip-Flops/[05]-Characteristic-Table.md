## Characteristic Table
A characteristics table in digital electronics defines the next state  (Qn+1) of a flip-flop based on its current state (Qn) and inputs (e.g J,K or S,R). 

### Characteristics Table for Common Flip-Flop

- SR Flip-Flop
  - 00: Hold (Qn+1 = Qn)
  - 01: Reset (Qn+1 = 0)
  - 10: Set (Qn+1 = 1)
  - 11: Invalid (-)
    
- JK Flip-Flop
  - 00: Hold (Qn+1 = Qn)
  - 01: Reset (Qn+1 = 0)
  - 10: Set (Qn+1 = 1)
  - 11: Toggle (Qn+1 = Q'n)
  
- D Flip-Flop
  - 0: Hold (Qn+1 = 0)
  - 1: Toggle (Qn+1 = 1)
 
- T Flip-Flop
  - 0: Hold (Qn+1 = Qn)
  - 1: Toggle (Qn+1 = Q'n )
