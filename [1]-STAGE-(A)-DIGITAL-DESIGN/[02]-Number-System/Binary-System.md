## Binary Number system
The Binary number system is a fundamental number system used in the digital electronics and computer systems.

Digits:
0 - OFF State

1 - ON State

### Example :-

(1 1 1 0 0 0)2

| - | - |  - | - | - | 

2⁵-2⁴-2³-2²-2¹-2⁰

It will be taken base '2' because binary have two characters.
