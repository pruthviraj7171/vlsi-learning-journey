## Logic Gates 

Logic gates perform logical operations on one of more binary inputs to produce a single output

## Types of Logic Gates

- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
_______________
