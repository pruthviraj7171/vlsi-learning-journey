## Binary To Hexadecimal 
Here conversation of binary to number

### For example
we have an binary number

1] 01001110111.110101010 (base 2)

where base '2' indicates Binary

The above binary number make a group of four 

0100,1111,0111,1101,0101,0000 (base 2) - Binary

  |----|----|----|----|----| 

  4.   F.   7.   D.   5.   0 (base 16) - Hexadecimal

The final value giver after conversation from binary to hexadecimal

--------------------------
| 4 F 7 . D 5 0 (base 16) |
--------------------------

010011110111.110101010 (base 2) = 4F7.D50 (base 16)
