## T Flip-Flop
A T flip flop stands for (Toggle) it is a fundamental sequencial logic circuit that changes its output state (toggles) on every active clock edge if its input (T) is high (1), or hold the current state if T is low (0). known as binary counter element, it is essentially modified  JK flip-flop with its J and K inputs tied together, offering a simple way to implement memory or frequency division.

### Truth Table

| CLK | T | Qn+1 |
|:---:|:-:|:----:|
|  0  | - |  Qn  |
|  1  | 0 |  Qn  |
|  1  | 1 |  Q'n |
