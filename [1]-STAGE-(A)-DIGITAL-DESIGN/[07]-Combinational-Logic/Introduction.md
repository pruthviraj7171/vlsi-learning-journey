## Combinational Logic 
The combinational logic is a digital circuit where output is depends only on present input.It does not depends only any past inputs because it has no memory element.

In combinational circuit whenever the inputs change the output also changes immediately according to the logic function impelemented in the circuit.

### working principle
The inputs are applied to the logic circuit include AND gates,OR gates,NOT gates,multiplexer,decoder,encoder,half adder, and full adders.
